# FIVEVAC-VACCINE-DEMO
Fivevac là một website quản lý tiêm chủng và đặt lịch tiêm vắc-xin được xây dựng bằng ngôn ngữ PHP và cơ sở dữ liệu MySQL. Ứng dụng giúp khách hàng đặt lịch tiêm và theo dõi lịch sử tiêm chủng một cách dễ dàng, đồng thời hỗ trợ nhân viên và quản trị viên quản lý tồn kho, hồ sơ, lịch hẹn và đơn hàng.   Giao diện Bootstrap 5 nên thân thiện.
